## 문제 1: 화학식 및 복합 단어 인식 실패

### 근본 원인 분석

`tfsi- emim+에 대해 전수조사`라는 입력이 실패하는 경로를 추적하면:

1. **`agent.py`의 `_extract_structure_query`**: 정규식 패턴들이 `+`와 `-`를 포함한 이온쌍 명칭을 정확히 추출하지 못합니다. 특히 `[a-zA-Z][a-zA-Z0-9_\-\+\(\), ]{1,80}` 같은 패턴에서 `tfsi- emim+`가 매칭되더라도, 뒤이은 noise word 제거 로직이 "에 대해"를 잘라낸 뒤 `tfsi- emim+`를 남기지만, 실제로는 한국어 조사 처리가 불완전합니다.

2. **`compute.py`의 `_fallback_extract_structure_query`**: `_normalize_text_token`이 `[^\w\s가-힣+\-]`로 필터링하여 `+`와 `-`는 살리지만, 정규식 패턴의 캡처 그룹에서 noise 제거 시 `re.sub(rf"\b{n}\b", ...)` 형태가 한국어 단어 경계에서 오작동합니다. "전수조사"는 noise 목록에 없어서 `tfsi- emim+전수조사`가 통째로 structure_query로 넘어갑니다.

3. **`pyscf_runner.py`의 `_resolve_structure_payload`**: 최종적으로 `MoleculeResolver.resolve_with_friendly_errors("tfsi- emim+전수조사")` 또는 유사한 더러운 문자열이 PubChem으로 전달되어 실패합니다.

4. **가장 핵심적인 문제**: `_prepare_payload`에서 `structure_query`, `xyz`, `atom_spec` 모두 없으면 즉시 HTTP 400 `"Structure not recognized."`를 던집니다. 파싱 자체가 실패하면 아무 계산도 시작되지 않습니다.

### 수정: `compute.py`의 `_fallback_extract_structure_query`

```python
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
# FILE: web/routes/compute.py
# FUNCTION: _fallback_extract_structure_query (전면 교체)
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

def _fallback_extract_structure_query(message: str) -> Optional[str]:
    """Extract a molecule/structure name from a free-form user message.

    Handles:
    - Korean alias lookup (물, 벤젠, ...)
    - Builtin library names (water, benzene, ...)
    - Ion-pair names with charges and spaces (tfsi- emim+, Li+ PF6-, ...)
    - Chemical formulas (H2O, CH3OH, C6H6, ...)
    - IUPAC / common English names (acetone, formaldehyde, ...)
    """
    if not message:
        return None
    if _extract_xyz_block(message):
        return None

    normalized = _normalize_text_token(message)

    # ── Step 1: Korean aliases (longest match first) ──
    for ko_name, en_name in sorted(
        _KO_STRUCTURE_ALIASES.items(), key=lambda x: len(x[0]), reverse=True
    ):
        if ko_name in normalized:
            return en_name

    # ── Step 2: Builtin library names ──
    structure_names = list(_iter_runner_structure_names())
    for name in sorted(structure_names, key=len, reverse=True):
        if _normalize_text_token(name) in normalized:
            return name

    # ── Step 3: Ion-pair / charged species pattern ──
    # Matches patterns like: "tfsi- emim+", "emim+ tfsi-", "Li+ PF6-",
    # "BF4- BMIM+", "Na+ Cl-", etc.
    # Also matches single ions: "tfsi-", "emim+"
    ion_pair_pat = re.compile(
        r'([A-Za-z][A-Za-z0-9]*[\+\-]\s*[A-Za-z][A-Za-z0-9]*[\+\-])',
        re.IGNORECASE,
    )
    m_ion = ion_pair_pat.search(message)
    if m_ion:
        return m_ion.group(1).strip()

    # Single ion with charge: "TFSI-", "EMIM+", "PF6-"
    single_ion_pat = re.compile(
        r'([A-Za-z][A-Za-z0-9]{1,20}[\+\-](?:\d*))',
        re.IGNORECASE,
    )
    # Try to find multiple ions separated by space/comma
    ions_found = single_ion_pat.findall(message)
    if len(ions_found) >= 2:
        return " ".join(ions_found[:2])
    if len(ions_found) == 1:
        return ions_found[0].strip()

    # ── Step 4: Chemical formula pattern ──
    # Matches: H2O, CH3OH, C6H6, CO2, NH3, C2H5OH, etc.
    formula_pat = re.compile(
        r'\b([A-Z][a-z]?(?:\d+)?(?:[A-Z][a-z]?(?:\d+)?){1,20})\b'
    )
    formulas = formula_pat.findall(message)
    # Filter out noise words that look like formulas
    noise_words = {
        "HOMO", "LUMO", "ESP", "SCF", "DFT", "HF", "MO",
        "Ha", "Hz", "OK", "No", "On", "Off", "In",
    }
    formulas = [f for f in formulas if f not in noise_words and len(f) >= 2]
    if formulas:
        # Heuristic: pick the one that looks most like a chemical formula
        # (has at least one uppercase letter followed by optional digits)
        chem_pat = re.compile(r'^[A-Z][a-z]?\d*(?:[A-Z][a-z]?\d*)+$')
        for f in formulas:
            if chem_pat.match(f):
                return f

    # ── Step 5: Korean sentence structure extraction ──
    # "tfsi- emim+에 대해 전수조사" → extract everything before Korean postpositions
    ko_postpos_pat = re.compile(
        r'([A-Za-z0-9\+\-\(\)\[\]\s,\.]{2,60}?)\s*'
        r'(?:에\s*대해|에\s*대한|의|에서|을|를|이|가|은|는|으로|로|와|과|하고|부터)',
        re.IGNORECASE,
    )
    m_ko = ko_postpos_pat.search(message)
    if m_ko:
        candidate = m_ko.group(1).strip().rstrip(" .,;:")
        if candidate and len(candidate) >= 2:
            return candidate

    # ── Step 6: General regex patterns ──
    patterns = [
        r"(?i)(?:for|of|on|about)\s+(?:the\s+)?([a-zA-Z][a-zA-Z0-9_\-\+\(\), ]{1,40})",
        r"(?i)([a-zA-Z][a-zA-Z0-9_\-\+\(\), ]{1,40})\s+(?:molecule|structure|system)",
        r"([가-힣A-Za-z0-9_\-\+\(\), ]+?)\s*(?:의|에\s*대한)?\s*(?:homo|lumo|esp|전하|구조|에너지|최적화|분석|전수조사|보여줘|해줘|계산)",
        r"([가-힣A-Za-z0-9_\-\+\(\), ]+?)\s+(?:분자|구조|이온쌍|이온)",
        r"(?i)(?:analyze|show|render|preview|compute|optimize|calculate)\s+(?:the\s+)?([a-zA-Z][a-zA-Z0-9_\-\+\(\), ]{1,40})",
    ]
    for pat in patterns:
        m = re.search(pat, message, re.IGNORECASE)
        if not m:
            continue
        candidate = _safe_str(m.group(1)).strip()

        # Remove noise words
        noise = [
            "homo", "lumo", "esp", "map", "orbital", "orbitals",
            "charge", "charges", "mulliken", "partial", "geometry",
            "optimization", "analysis", "of", "about", "for",
            "보여줘", "해줘", "계산", "전수조사", "분석",
        ]
        for n in noise:
            candidate = re.sub(rf"\b{re.escape(n)}\b", " ", candidate, flags=re.I)

        # Strip Korean postpositions from the end
        ko_suffixes = [
            "에 대해 전수조사", "에 대해", "에대한", "에 대한",
            "이온쌍", "의", "분자", "구조", "계산", "해줘",
            "보여줘", "전수조사", "분석",
        ]
        for suffix in sorted(ko_suffixes, key=len, reverse=True):
            if candidate.endswith(suffix):
                candidate = candidate[: -len(suffix)].strip()

        candidate = re.sub(r"\s+", " ", candidate).strip()
        if not candidate:
            continue

        candidate_norm = _normalize_text_token(candidate)
        if candidate_norm in _KO_STRUCTURE_ALIASES:
            return _KO_STRUCTURE_ALIASES[candidate_norm]
        for name in structure_names:
            if _normalize_text_token(name) == candidate_norm:
                return name
        return candidate

    return None
```

### 수정: `agent.py`의 `_extract_structure_query`

```python
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
# FILE: llm/agent.py
# METHOD: QCVizAgent._extract_structure_query (전면 교체)
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    def _extract_structure_query(self, text: str) -> Optional[str]:
        # ── 1. Pasted XYZ block ──
        if len(re.findall(r"\n", text)) >= 2 and re.search(
            r"^[A-Z][a-z]?\s+-?\d", text, re.M
        ):
            return text.strip()

        # ── 2. Quoted string ──
        quoted = re.findall(r'"([^"]+)"|\'([^\']+)\'', text)
        if quoted:
            first = quoted[0][0] or quoted[0][1]
            if first.strip():
                return first.strip()

        # ── 3. Ion-pair detection (before general patterns) ──
        # "tfsi- emim+", "EMIM+ TFSI-", "Li+ PF6-", "BF4- bmim+"
        ion_pair_pat = re.compile(
            r'([A-Za-z][A-Za-z0-9]*[\+\-]\s*[A-Za-z][A-Za-z0-9]*[\+\-])',
            re.IGNORECASE,
        )
        m_ion = ion_pair_pat.search(text)
        if m_ion:
            return m_ion.group(1).strip()

        # Multiple single ions
        single_ion_pat = re.compile(
            r'([A-Za-z][A-Za-z0-9]{1,20}[\+\-])',
            re.IGNORECASE,
        )
        ions = single_ion_pat.findall(text)
        if len(ions) >= 2:
            return " ".join(ions[:2])

        # ── 4. Korean postposition extraction ──
        ko_pat = re.compile(
            r'([A-Za-z0-9\+\-\(\)\[\]\s,\.]{2,60}?)\s*'
            r'(?:에\s*대해|에\s*대한|의|에서|을|를|이|가|은|는)',
            re.IGNORECASE,
        )
        m_ko = ko_pat.search(text)
        if m_ko:
            candidate = m_ko.group(1).strip().rstrip(" .,;:")
            if candidate and len(candidate) >= 2:
                return candidate

        # ── 5. General English patterns ──
        patterns = [
            r"(?i)(?:for|of|on|about)\s+(?:the\s+)?([a-zA-Z][a-zA-Z0-9_\-\+\(\), ]{1,80})",
            r"(?i)([a-zA-Z][a-zA-Z0-9_\-\+\(\), ]{1,80})\s+(?:molecule|structure|system)",
            r"([가-힣A-Za-z0-9_\-\+\(\), ]+?)\s*(?:의|에\s*대한)?\s*(?:homo|lumo|esp|전하|구조|에너지|최적화|분석|전수조사|보여줘|해줘|계산)",
            r"([가-힣A-Za-z0-9_\-\+\(\), ]+?)\s+(?:분자|구조|이온쌍|이온)",
            r"(?i)(?:analyze|compute|calculate|show|render|visualize|optimize)\s+(?:the\s+)?([A-Za-z0-9_\-\+\(\), ]{2,80})",
        ]
        for pat in patterns:
            m = re.search(pat, text, re.I)
            if m:
                candidate = m.group(1).strip(" .,:;")
                candidate = re.split(
                    r"\b(using|with|at|in|and show|and render|method|basis|charge|multiplicity|spin|preset)\b",
                    candidate,
                    maxsplit=1,
                    flags=re.I,
                )[0].strip(" .,:;")

                # Filter Korean noise
                ko_noise = [
                    "에 대해 전수조사", "에 대해", "에대한", "에 대한",
                    "의", "분자", "구조", "계산", "해줘", "보여줘",
                    "전수조사", "분석",
                ]
                for noise in sorted(ko_noise, key=len, reverse=True):
                    if candidate.endswith(noise):
                        candidate = candidate[: -len(noise)].strip(" .,:;")

                # Remove English noise
                en_noise = [
                    "homo", "lumo", "orbital", "orbitals", "esp", "map",
                    "charge", "charges", "mulliken", "analysis",
                ]
                for n in en_noise:
                    candidate = re.sub(rf"\b{re.escape(n)}\b", " ", candidate, flags=re.I)
                candidate = re.sub(r"\s+", " ", candidate).strip()

                if candidate and len(candidate) >= 2:
                    return candidate

        # ── 6. Common molecule names ──
        common = [
            "water", "methane", "ammonia", "benzene", "ethanol",
            "acetone", "formaldehyde", "carbon dioxide", "co2",
            "nh3", "h2o", "caffeine", "naphthalene", "pyridine", "phenol",
        ]
        lower = text.lower()
        for name in common:
            if name in lower:
                return name

        return None
```

### 수정: `compute.py`의 `_prepare_payload` (더 나은 에러 메시지 + 유연한 처리)

```python
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
# FILE: web/routes/compute.py
# FUNCTION: _prepare_payload (수정된 부분만)
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

def _prepare_payload(payload: Mapping[str, Any]) -> Dict[str, Any]:
    data = dict(payload or {})
    raw_message = _extract_message(data)

    if raw_message and not data.get("planner_applied"):
        plan = _safe_plan_message(raw_message, data)
        data = _merge_plan_into_payload(data, plan, raw_message=raw_message)

    data["job_type"] = _normalize_job_type(data.get("job_type"), data.get("planner_intent"))
    data["method"] = _safe_str(
        data.get("method") or getattr(pyscf_runner, "DEFAULT_METHOD", "B3LYP"),
        getattr(pyscf_runner, "DEFAULT_METHOD", "B3LYP"),
    )
    data["basis"] = _safe_str(
        data.get("basis") or getattr(pyscf_runner, "DEFAULT_BASIS", "def2-SVP"),
        getattr(pyscf_runner, "DEFAULT_BASIS", "def2-SVP"),
    )
    data["charge"] = _safe_int(data.get("charge"), 0) or 0
    data["multiplicity"] = _safe_int(data.get("multiplicity"), 1) or 1

    if data.get("esp_preset"):
        data["esp_preset"] = _normalize_esp_preset(data.get("esp_preset"))

    if not data.get("xyz"):
        xyz_block = _extract_xyz_block(raw_message)
        if xyz_block:
            data["xyz"] = xyz_block

    if not data.get("structure_query") and not data.get("xyz") and not data.get("atom_spec"):
        fallback = _fallback_extract_structure_query(raw_message)
        if fallback:
            data["structure_query"] = fallback

    # ── IMPROVED: Last-resort extraction for ion pairs / chemical names ──
    # If still nothing, try to extract ANY plausible chemical token from the message
    if not (data.get("structure_query") or data.get("xyz") or data.get("atom_spec")):
        if raw_message:
            # Try aggressive extraction: any word with +/- (ion), or capitalized
            # multi-letter tokens that might be molecule names
            aggressive_ion = re.findall(
                r'[A-Za-z][A-Za-z0-9]*[\+\-]', raw_message
            )
            if aggressive_ion:
                data["structure_query"] = " ".join(aggressive_ion[:3])

    if data["job_type"] not in {"resolve_structure"}:
        if not (data.get("structure_query") or data.get("xyz") or data.get("atom_spec")):
            raise HTTPException(
                status_code=400,
                detail=(
                    "Structure not recognized. Could not extract a molecule name "
                    "from your message. Please provide a molecule name (e.g., 'water', "
                    "'benzene'), a chemical formula (e.g., 'H2O', 'C6H6'), SMILES, "
                    "or paste XYZ coordinates directly."
                ),
            )

    return data
```

## 문제 2: WebSocket Keepalive Timeout 에러

### 근본 원인 분석

현재 코드에 WebSocket endpoint 정의가 보이지 않습니다(compute.py의 라우터에는 REST endpoint만 있음). 그러나 `chat.js`가 `ws://host/api/ws/chat`에 연결하므로, 어딘가에 WebSocket 핸들러가 있을 것입니다. 문제는:

1. **서버 측**: 장시간 계산(SCF 등) 중 비동기 루프가 블로킹되어 ping/pong 프레임 처리가 지연됩니다.
2. **클라이언트 측**: `chat.js`에 ping/pong 응답 로직이 없습니다. 서버가 ping을 보내면 브라우저 WebSocket API가 자동으로 pong을 응답하지만, 서버의 `ping_timeout`이 너무 짧거나 서버 이벤트 루프가 막혀 있으면 timeout이 발생합니다.

### 해결: WebSocket Endpoint 코드 (신규 또는 교체)

서버 측에 WebSocket 라우터가 별도 파일에 있을 수 있지만, 여기서는 완전한 방어적 구현을 제공합니다:

```python

# FILE: web/routes/ws_chat.py (신규 생성 또는 기존 WS 핸들러 교체)

# ── Keepalive settings ──
WS_PING_INTERVAL = 25  # seconds between server→client pings
WS_PING_TIMEOUT = 60   # seconds to wait for pong (generous for slow networks)
WS_POLL_INTERVAL = 0.3 # seconds between job status polls


async def _safe_send_json(ws: WebSocket, data: Dict[str, Any]) -> bool:
    """Send JSON to WebSocket, returning False if the connection is closed."""
    try:
        if ws.client_state == WebSocketState.CONNECTED:
            await ws.send_json(_json_safe(data))
            return True
    except Exception as exc:
        logger.debug("WS send failed: %s", exc)
    return False


async def _keepalive_ping(ws: WebSocket, stop_event: asyncio.Event) -> None:
    """Periodically send application-level ping to keep the connection alive.

    This is separate from the WebSocket protocol-level ping/pong.
    It prevents proxies (nginx, cloudflare) from closing idle connections.
    """
    while not stop_event.is_set():
        try:
            await asyncio.wait_for(stop_event.wait(), timeout=WS_PING_INTERVAL)
            break  # stop_event was set
        except asyncio.TimeoutError:
            pass  # normal: timeout means it's time to ping

        ok = await _safe_send_json(ws, {"type": "pong", "ts": time.time()})
        if not ok:
            break


async def _poll_and_stream_job(
    ws: WebSocket,
    job_id: str,
    stop_event: asyncio.Event,
) -> None:
    """Poll a job's status and stream progress/result events to the client."""
    last_event_count = 0
    last_progress = -1.0

    while not stop_event.is_set():
        snap = JOB_MANAGER.get(
            job_id,
            include_result=True,
            include_events=True,
        )
        if snap is None:
            await _safe_send_json(ws, {
                "type": "error",
                "job_id": job_id,
                "message": "Job not found",
            })
            return

        # Stream new events
        events = snap.get("events") or []
        if len(events) > last_event_count:
            for evt in events[last_event_count:]:
                evt_type = evt.get("type", "")
                progress = None
                if evt.get("data") and isinstance(evt["data"], dict):
                    progress = evt["data"].get("progress")

                # Avoid duplicate progress messages
                if progress is not None and progress == last_progress:
                    continue
                if progress is not None:
                    last_progress = progress

                await _safe_send_json(ws, {
                    "type": "job_progress",
                    "job_id": job_id,
                    "status": snap.get("status", "running"),
                    "progress": progress,
                    "step": evt.get("data", {}).get("step", ""),
                    "message": evt.get("message", ""),
                })
            last_event_count = len(events)

        # Check for terminal state
        status = snap.get("status", "")
        if status in TERMINAL_STATES:
            if status == "completed" and snap.get("result"):
                await _safe_send_json(ws, {
                    "type": "result",
                    "job_id": job_id,
                    "status": "completed",
                    "result": snap["result"],
                })
            elif status in {"failed", "error"}:
                err = snap.get("error") or {}
                await _safe_send_json(ws, {
                    "type": "error",
                    "job_id": job_id,
                    "status": "failed",
                    "message": err.get("message", "Computation failed"),
                    "error": err,
                })
            return

        # Non-blocking poll wait
        try:
            await asyncio.wait_for(stop_event.wait(), timeout=WS_POLL_INTERVAL)
            break
        except asyncio.TimeoutError:
            pass


@router.websocket("/chat")
async def ws_chat_endpoint(ws: WebSocket) -> None:
    """WebSocket endpoint for chat-based computation.

    Key defensive measures against keepalive timeouts:
    1. Application-level keepalive pings every 25s
    2. All blocking work (SCF computation) runs in ThreadPoolExecutor via JOB_MANAGER
    3. Generous ping_timeout of 60s
    4. Graceful cleanup on any disconnection
    """
    await ws.accept()
    logger.info("WS client connected: %s", ws.client)

    stop_event = asyncio.Event()
    keepalive_task = asyncio.create_task(_keepalive_ping(ws, stop_event))

    # Send ready signal
    await _safe_send_json(ws, {"type": "ready", "message": "Connected to QCViz-MCP"})

    try:
        while True:
            # Use a timeout so we can check connection health
            try:
                raw = await asyncio.wait_for(ws.receive_text(), timeout=WS_PING_TIMEOUT)
            except asyncio.TimeoutError:
                # Client hasn't sent anything; send a ping to check liveness
                ok = await _safe_send_json(ws, {"type": "pong", "ts": time.time()})
                if not ok:
                    break
                continue

            # Parse incoming message
            try:
                msg = json.loads(raw)
            except (json.JSONDecodeError, TypeError):
                await _safe_send_json(ws, {
                    "type": "error",
                    "message": "Invalid JSON",
                })
                continue

            msg_type = _safe_str(msg.get("type"), "chat").lower()

            # Handle client pings
            if msg_type in {"ping", "heartbeat", "keepalive"}:
                await _safe_send_json(ws, {"type": "pong", "ts": time.time()})
                continue

            # Handle chat messages
            if msg_type in {"chat", "message", "compute", "submit"}:
                user_message = _extract_message(msg)
                session_id = _extract_session_id(msg)

                if not user_message:
                    await _safe_send_json(ws, {
                        "type": "error",
                        "message": "Empty message",
                    })
                    continue

                # Submit job (non-blocking — runs in thread pool)
                try:
                    payload = dict(msg)
                    payload["message"] = user_message
                    if session_id:
                        payload["session_id"] = session_id

                    snap = JOB_MANAGER.submit(payload)
                    job_id = snap["job_id"]

                    # Notify client of submission
                    await _safe_send_json(ws, {
                        "type": "job_submitted",
                        "job_id": job_id,
                        "status": "queued",
                        "message": "Job submitted",
                        "molecule_name": snap.get("molecule_name", ""),
                        "method": snap.get("method", ""),
                        "basis_set": snap.get("basis_set", ""),
                    })

                    # Stream progress in background (non-blocking)
                    asyncio.create_task(
                        _poll_and_stream_job(ws, job_id, stop_event)
                    )

                except Exception as exc:
                    logger.exception("Job submission failed")
                    await _safe_send_json(ws, {
                        "type": "error",
                        "message": str(exc),
                    })

    except WebSocketDisconnect:
        logger.info("WS client disconnected normally")
    except Exception as exc:
        logger.warning("WS error: %s\n%s", exc, traceback.format_exc())
    finally:
        # Clean shutdown
        stop_event.set()
        keepalive_task.cancel()
        try:
            await keepalive_task
        except (asyncio.CancelledError, Exception):
            pass
        logger.info("WS session cleaned up")
```

```javascript
// ── Client-side keepalive interval ──
var CLIENT_PING_INTERVAL = 20000; // 20 seconds
var clientPingTimer = null;

function startClientPing() {
  stopClientPing();
  clientPingTimer = setInterval(function () {
    safeSendWs({ type: "ping", ts: Date.now() });
  }, CLIENT_PING_INTERVAL);
}

function stopClientPing() {
  if (clientPingTimer) {
    clearInterval(clientPingTimer);
    clientPingTimer = null;
  }
}

function connectWS() {
  // Clear old handlers to prevent ghost callbacks
  if (state.ws) {
    if (
      state.ws.readyState === WebSocket.OPEN ||
      state.ws.readyState === WebSocket.CONNECTING
    )
      return;
    state.ws.onopen = null;
    state.ws.onclose = null;
    state.ws.onerror = null;
    state.ws.onmessage = null;
    state.ws = null;
  }

  stopClientPing();

  try {
    state.ws = new WebSocket(buildWsUrl());
  } catch (e) {
    setWsUI(false);
    scheduleReconnect();
    return;
  }

  state.ws.onopen = function () {
    if (this !== state.ws) return;
    setWsUI(true);
    state.reconnectAttempts = 0;
    startClientPing(); // ← Start keepalive pings
    console.log(
      "%c[WS] Connected",
      "background:#22c55e;color:white;padding:2px 6px;border-radius:3px;",
    );
  };

  state.ws.onclose = function (event) {
    if (this !== state.ws) return;
    setWsUI(false);
    stopClientPing(); // ← Stop keepalive pings
    console.log(
      "%c[WS] Closed (code=" +
        event.code +
        ", reason=" +
        (event.reason || "none") +
        ")",
      "background:#ef4444;color:white;padding:2px 6px;border-radius:3px;",
    );
    scheduleReconnect();
  };

  state.ws.onerror = function (event) {
    if (this !== state.ws) return;
    setWsUI(false);
    console.warn("[WS] Error event", event);
  };

  state.ws.onmessage = function (event) {
    if (this !== state.ws) return;
    var data;
    try {
      data = JSON.parse(event.data);
    } catch (_) {
      return;
    }
    handleServerEvent(data);
  };
}

function scheduleReconnect() {
  if (state.reconnectTimer) return;
  if (state.reconnectAttempts >= state.maxReconnect) {
    console.warn(
      "[WS] Max reconnection attempts reached (" + state.maxReconnect + ")",
    );
    // Reset after a long delay to allow future reconnections
    setTimeout(function () {
      state.reconnectAttempts = 0;
    }, 60000);
    return;
  }
  var delay = Math.min(1000 * Math.pow(2, state.reconnectAttempts), 30000);
  state.reconnectAttempts++;
  console.log(
    "[WS] Reconnecting in " +
      delay +
      "ms (attempt " +
      state.reconnectAttempts +
      ")",
  );
  state.reconnectTimer = setTimeout(function () {
    state.reconnectTimer = null;
    connectWS();
  }, delay);
}
```

### 수정: 높이가 초기 로딩 시 틀어지는 문제 해결

이 문제의 근본 원인은 두 가지입니다:

1. **`html, body { height: 100vh }`와 `.app-shell`의 높이 계산 불일치**: `app-shell`이 `height: 100vh`이면서 `padding: 18px`과 `gap: 18px`을 가지고 있는데, 내부 요소들이 `100vh`를 초과하면 overflow가 발생합니다. `flex: 1`인 dashboard가 실제로 사용할 수 있는 높이는 `100vh - topbar높이 - sessionTabs높이 - padding*2 - gap*2`입니다.

2. **`.viewer-container { min-height: 300px }`**: 이 최소 높이가 초기 상태에서 다른 패널들의 높이를 밀어냅니다.

위 CSS에서 이미 수정했지만, 핵심 변경을 강조합니다:

```css

}

/* 2. Topbar: fixed height, don't grow */

  min-height: 0; /* ← critical for flex/grid children */
}
```

### 추가: Right Column Divider 리사이즈 기능 (app.js에 추가)

```javascript
// ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
// FILE: web/static/app.js
// SECTION: Right column divider resize (파일 끝에 추가)
// ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

/* ─── Right Column Resizable Divider ─── */
(function () {
  var $divider = document.getElementById("rightColDivider");
  var $history = document.getElementById("subPanelHistory");
  var $rightCol = document.querySelector(".panel--right-col");
  if (!$divider || !$history || !$rightCol) return;

  var dragging = false;
  var startY = 0;
  var startH = 0;

  $divider.addEventListener("mousedown", function (e) {
    e.preventDefault();
    dragging = true;
    startY = e.clientY;
    startH = $history.offsetHeight;
    document.body.style.cursor = "row-resize";
    document.body.style.userSelect = "none";
  });

  document.addEventListener("mousemove", function (e) {
    if (!dragging) return;
    var delta = e.clientY - startY;
    var totalH = $rightCol.offsetHeight;
    var newH = Math.max(100, Math.min(totalH - 150, startH + delta));
    $history.style.maxHeight = newH + "px";
    $history.style.flex = "0 0 " + newH + "px";
  });

  document.addEventListener("mouseup", function () {
    if (!dragging) return;
    dragging = false;
    document.body.style.cursor = "";
    document.body.style.userSelect = "";
  });
})();
```
