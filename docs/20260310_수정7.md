## Part 1: 원인 분석 리포트

### 1‑A. viewer.js — 발견된 결함 총 14개

**결함 #1: `renderOrbital` — 미선언 변수 `scheme` 참조 (ReferenceError)**
189번째 줄 부근에서 음의 로브 색상을 `scheme.orbNegative`로 참조하지만, `scheme`이라는 변수는 이 함수 스코프에 선언된 적이 없습니다. `getCurrentColorScheme()`의 반환값을 `var scheme`에 할당하는 코드가 누락되어 있습니다. 이 에러가 `try/catch`에 잡혀 조용히 무시되므로, 오비탈 렌더링이 실패해도 사용자에게는 빈 화면만 보입니다.

**결함 #2: `switchVizMode` — `_vizSwitchLock` 이중 선언**
`var _vizSwitchLock = false;`가 두 번 연속 선언되어 있습니다. 기능적으로는 무해하지만, 코드 품질 문제이며 이전 패치가 무분별했음을 보여줍니다.

**결함 #3: `switchVizMode` — `state.mode === newMode` 체크 누락**
`switchVizMode` 진입부에 `if (state.mode === newMode) return;` 가드가 없습니다. 이전 버전에서 Color Scheme 변경 시 `state.mode = "none"` → `switchVizMode(currentMode)` 트릭을 쓰기 위해 의도적으로 제거한 것으로 보이지만, 이로 인해 같은 모드를 연속 클릭하면 불필요한 재렌더가 발생합니다.

**결함 #4: Color Scheme change 핸들러 — `switchVizMode` 경로의 lock 충돌**
`init()` 내 Color Scheme change 핸들러가 `state.mode = "none"` → `switchVizMode(currentMode)`를 호출합니다. 하지만 `switchVizMode`는 30ms `setTimeout` 안에서 렌더링하고 `_vizSwitchLock`을 30ms간 잠그므로, 빠르게 스킴을 바꾸면 lock에 걸려 아무 반응이 없습니다.

**결함 #5: `switchVizMode`가 `clearViewer` 대신 `removeAllSurfaces`만 호출**
`switchVizMode`는 표면만 제거하고 모델은 유지하지만, `tryRenderCachedOrbital`은 내부에서 `clearViewer`(모델 포함 전체 제거) → `addMoleculeModel`을 합니다. 이 두 경로의 모델 상태가 일치하지 않아, 캐시 히트 시 모델이 이중으로 추가되거나 레이블이 꼬입니다.

**결함 #6: 이벤트 리스너 이중 등록**
`$selectOrbital`, `$sliderIso`, `$sliderOp`, `$btnLabels`, `$btnReset`, `$btnScreenshot`, `$btnFullscreen` — 이 7개 DOM 요소에 대한 이벤트 리스너가 IIFE 상단의 인라인 코드와 `init()` → `bindStyleButtons()` / `bindReactiveControls()` 양쪽에서 모두 등록됩니다. 결과적으로 슬라이더 변경 시 `renderOrbital`이 2번 호출되고, 라벨 토글이 2번 실행되어 토글이 취소됩니다.

**결함 #7: `bindStyleButtons()` 내 `clearLabels` 미정의**
`bindStyleButtons`의 라벨 토글 핸들러에서 `clearLabels()`를 호출하지만, 이 함수는 어디에도 정의되어 있지 않습니다. `ReferenceError`가 발생하여 라벨 토글이 완전히 먹통이 됩니다.

**결함 #8: `init()` 내 `loadViewerSnapshot` 미정의**
`init()`의 `ensureViewer().then(...)` 안에서 `loadViewerSnapshot(activeJobId)`를 호출하지만, 이 함수는 정의된 적이 없습니다. `restoreViewerSnapshot`이 올바른 이름인데, 오타가 발생했습니다. 이 `ReferenceError`가 Promise chain을 죽이고, 그 뒤의 `dismissLoader()` 호출까지 건너뛰어 **무한 로딩**을 유발합니다. `.catch`에서 `dismissLoader()`를 호출하긴 하지만, `catch` 블록 자체에 다른 에러가 겹치면 로더가 영영 안 사라집니다.

**결함 #9: `handleResultSwitched` — `loadMoleculeXyz` 미정의**
세션 탭 전환 이벤트 핸들러에서 `loadMoleculeXyz(xyz)`를 호출하지만, 이 함수는 어디에도 정의되어 있지 않습니다. 이전 패치에서 제안만 하고 실제 구현을 빠뜨렸습니다. 세션 탭 전환 시 즉시 `ReferenceError`가 발생합니다.

**결함 #10: `showControls` 함수 중복 정의**
`showControls`가 두 번 정의되어 있습니다. JavaScript에서 같은 이름의 함수 선언이 두 번 나오면 후자가 전자를 덮어씁니다. 첫 번째 정의의 슬라이더 범위 조정 로직이 사라진 게 아니라 우연히 두 번째에도 포함되어 있지만, 이런 중복은 유지보수에 치명적입니다.

**결함 #11: `colorScheme` 상태 미초기화**
`state` 객체에 `colorScheme` 프로퍼티가 없습니다. `getCurrentColorScheme()`이 `COLOR_SCHEMES[undefined]`를 조회하여 `undefined`가 반환되고, `|| COLOR_SCHEMES.classic` 폴백으로 항상 classic만 됩니다. 스킴 변경 후 `state.colorScheme`에 값이 할당되면 정상 동작하지만, 초기 렌더링 시에는 반드시 fallback을 탑니다.

**결함 #12: `espDensityIso` 상태 미초기화**
`state.espDensityIso`가 선언되어 있지 않아 `undefined`이 됩니다. `|| 0.001` 폴백으로 동작은 하지만, 명시적 초기화가 필요합니다.

**결함 #13: `dismissLoader` 3초 타이머가 IIFE 상단에서 즉시 실행**
`setTimeout(dismissLoader, 3000);`이 모듈 로드 직후 실행됩니다. 3Dmol.js 로딩이 3초 이상 걸리면 로더가 사라진 뒤에도 뷰어가 아직 준비되지 않아 빈 화면이 됩니다. 반대로 3초 안에 모든 초기화가 끝나면 불필요한 타이머입니다.

**결함 #14: `renderESP`의 else 분기 — `range` 변수 미정의**
`renderESP` 함수의 `else if (espB64)` 분기에서 `range` 변수가 정의되지 않은 채 `scheme2.reverse ? 0.05 : -0.05`로 하드코딩되어 있습니다. `result.esp_auto_range_au`를 읽어야 하는데 누락되었습니다.

### 1‑B. app.js — 발견된 결함 3개

**결함 #15: `loadViewerSnapshot` 연쇄 실패**
viewer.js의 `init()`이 `loadViewerSnapshot`에서 터지면, app.js가 `jobs:changed` 이벤트를 받아도 세션 탭 렌더링이 viewer 상태와 동기화되지 않습니다.

**결함 #16: Session Tabs 시스템 중복**
`results.js`에 `sessionResults` 배열 + `renderSessionTabs`가 있고, `app.js`에도 `renderSessionTabs`가 있습니다. 두 시스템이 독립적으로 동작하며 서로 간섭합니다.

**결함 #17: History fetch 후 결과 복원 누재**
`fetchHistory()`가 job 목록을 가져와 `upsertJob`하지만, 마지막 활성 job의 결과를 자동으로 뷰어에 로드하지 않습니다. 새로고침 후 히스토리 목록은 보이지만 뷰어와 결과 패널은 비어있습니다.

### 1‑C. style.css — 발견된 결함 2개

**결함 #18: 뷰어 패널 찌그러짐**
`.dashboard` 그리드의 행 비율이 `minmax(450px, 1.8fr) minmax(200px, 0.6fr)`인데, 하단 패널 내용이 200px을 초과하면 `0.6fr`이 늘어나면서 상단 뷰어를 압축합니다. `minmax`의 최대값이 `fr` 단위라서 내용에 따라 유동적으로 변합니다.

**결함 #19: `.session-tab` 스타일 충돌**
`.session-tab-bar` 안의 탭과 `.session-tabs-container` 안의 탭이 같은 `.session-tab` 클래스를 공유하면서, 모서리 반경(`border-radius: 6px 6px 0 0` vs `border-radius: var(--radius-full)`)과 패딩이 충돌합니다.

  function addLabels(viewer, result) {
    var atoms = result.atoms || [];
    if (!atoms.length) return;

    var isDark = document.documentElement.getAttribute("data-theme") === "dark";
    var charges = result.mulliken_charges || result.lowdin_charges || [];

    var maxAbs = 0;
    for (var k = 0; k < charges.length; k++) {
      var cv = charges[k];
      var cval = cv != null && typeof cv === "object" ? cv.charge : cv;
      if (cval != null && isFinite(cval) && Math.abs(cval) > maxAbs)
        maxAbs = Math.abs(cval);
    }
    if (maxAbs < 0.001) maxAbs = 1;

    atoms.forEach(function (a, i) {
      var el = a.element || a.symbol || a[0] || "";
      if (!el) return;

      var rawCharge = charges[i];
      var chargeVal = null;
      if (rawCharge != null) {
        chargeVal =
          typeof rawCharge === "object" ? rawCharge.charge : rawCharge;
        if (chargeVal != null && !isFinite(chargeVal)) chargeVal = null;
      }

      var labelText = el;
      if (chargeVal != null) {
        labelText +=
          " (" + (chargeVal >= 0 ? "+" : "") + chargeVal.toFixed(3) + ")";
      }

      var bgColor, fontColor, borderColor;
      if (chargeVal != null && Math.abs(chargeVal) > 0.005) {
        var alpha = 0.25 + Math.min(Math.abs(chargeVal) / maxAbs, 1.0) * 0.55;
        if (chargeVal > 0) {
          bgColor = "rgba(59,130,246," + alpha.toFixed(2) + ")";
          fontColor = isDark ? "#dbeafe" : "#1e3a5f";
          borderColor = "rgba(59,130,246,0.4)";
        } else {
          bgColor = "rgba(239,68,68," + alpha.toFixed(2) + ")";
          fontColor = isDark ? "#fee2e2" : "#7f1d1d";
          borderColor = "rgba(239,68,68,0.4)";
        }
      } else {
        fontColor = isDark ? "white" : "#333";
        bgColor = isDark ? "rgba(0,0,0,0.5)" : "rgba(255,255,255,0.7)";
        borderColor = isDark ? "rgba(255,255,255,0.1)" : "rgba(0,0,0,0.1)";
      }

      viewer.addLabel(labelText, {
        position: {
          x: a.x != null ? a.x : a[1] || 0,
          y: a.y != null ? a.y : a[2] || 0,
          z: a.z != null ? a.z : a[3] || 0,
        },
        fontSize: 11,
        fontColor: fontColor,
        backgroundColor: bgColor,
        borderColor: borderColor,
        borderThickness: 1,
        backgroundOpacity: 0.85,
        alignment: "center",
        showBackground: true,
      });
    });
  }

  function clearViewer(viewer) {
    if (!viewer) return;
    viewer.removeAllModels();
    viewer.removeAllSurfaces();
    viewer.removeAllLabels();
    if (typeof viewer.removeAllShapes === "function") viewer.removeAllShapes();
    state.model = null;
  }

  function addMoleculeModel(viewer, result) {
    var xyz = getXyz(result);
    if (!xyz) return false;
    state.model = viewer.addModel(xyz, "xyz");
    applyStyle(viewer, state.style);
    return true;
  }

  /**
   * 오비탈 표면 추가 (viewer에 직접)
   * FIX #1: scheme 변수를 반드시 getScheme()으로 획득
   */
  function addOrbitalSurfaces(viewer, cubeString) {
    if (!cubeString) return;
    var scheme = getScheme();
    var vol = new window.$3Dmol.VolumeData(cubeString, "cube");
    viewer.addIsosurface(vol, {
      isoval: state.isovalue,
      color: scheme.orbPositive,
      alpha: state.opacity,
      smoothness: 3,
      wireframe: false,
    });
    viewer.addIsosurface(vol, {
      isoval: -state.isovalue,
      color: scheme.orbNegative,
      alpha: state.opacity,
      smoothness: 3,
      wireframe: false,
    });
  }

  /**
   * ESP 표면 추가
   * FIX #14: range 변수를 반드시 result에서 읽음
   */
  function addESPSurface(viewer, result) {
    var espB64 = findCubeB64(result, "esp");
    var densB64 = findCubeB64(result, "density");
    var espStr = safeAtob(espB64);
    if (!espStr) return;

    var scheme = getScheme();
    var range = result.esp_auto_range_au || 0.05;
    var minVal = scheme.reverse ? range : -range;
    var maxVal = scheme.reverse ? -range : range;
    var grad = createGradient(scheme.espGradient, minVal, maxVal);

    var espVol = new window.$3Dmol.VolumeData(espStr, "cube");

    if (densB64) {
      var densStr = safeAtob(densB64);
      if (densStr) {
        var densVol = new window.$3Dmol.VolumeData(densStr, "cube");
        viewer.addIsosurface(densVol, {
          isoval: state.espDensityIso,
          color: "white",
          alpha: state.opacity,
          smoothness: 1,
          voldata: espVol,
          volscheme: grad,
        });
        return;
      }
    }
    // ESP만 있는 경우
    viewer.addIsosurface(espVol, {
      isoval: state.espDensityIso,
      alpha: state.opacity,
      smoothness: 3,
      volscheme: grad,
    });
  }

  /* ─────────────────────────────────────────
     HIGH-LEVEL RENDERERS
     ───────────────────────────────────────── */
  function renderMolecule(result) {
    return ensureViewer().then(function (viewer) {
      clearViewer(viewer);
      addMoleculeModel(viewer, result);
      if (state.showLabels) addLabels(viewer, result);
      viewer.zoomTo();
      viewer.render();
      state.mode = "molecule";
      showControls("molecule");
      hideLegend();
    });
  }

  function renderOrbital(result) {
    return ensureViewer().then(function (viewer) {
      var oldXyz = state.result ? getXyz(state.result) : null;
      var newXyz = getXyz(result);
      var isNew = oldXyz !== newXyz;

      clearViewer(viewer);
      addMoleculeModel(viewer, result);

      var cubeB64 = findCubeB64(result, "orbital");
      var cubeStr = safeAtob(cubeB64);
      if (cubeStr) {
        addOrbitalSurfaces(viewer, cubeStr);
        if (!state.model) {
          state.model = viewer.addModel(cubeStr, "cube");
          applyStyle(viewer, state.style);
        }
      }

      if (state.showLabels && state.model) addLabels(viewer, result);
      if (isNew) viewer.zoomTo();
      viewer.render();
      state.mode = "orbital";
      showControls("orbital");
      showOrbitalLegend();
      populateOrbitalSelector(result);
    });
  }

  function renderESP(result) {
    return ensureViewer().then(function (viewer) {
      var oldXyz = state.result ? getXyz(state.result) : null;
      var newXyz = getXyz(result);
      var isNew = oldXyz !== newXyz;

      clearViewer(viewer);
      addMoleculeModel(viewer, result);

      try {
        addESPSurface(viewer, result);
      } catch (e) {
        console.error("[Viewer] ESP render error:", e);
      }

      if (state.showLabels && state.model) addLabels(viewer, result);
      if (isNew) viewer.zoomTo();
      viewer.render();
      state.mode = "esp";
      showControls("esp");
      showESPLegend();
    });
  }

      : "linear-gradient(90deg,#ef4444,#ffffff,#3b82f6)";
  }

  /* ─────────────────────────────────────────
     ORBITAL SELECTOR
     ───────────────────────────────────────── */
  function populateOrbitalSelector(result) {
    if (!dom.$selectOrbital || !result) return;
    var orbitals = result.orbitals || [];
    var moE = result.mo_energies || [];
    var moO = result.mo_occupations || [];
    dom.$selectOrbital.innerHTML = "";

    if (orbitals.length > 0) {
      var info =
        (result.visualization && result.visualization.orbital_info) ||
        result.orbital_info ||
        {};
      var currentIdx =
        info.orbital_index != null
          ? info.orbital_index
          : result.selected_orbital
            ? result.selected_orbital.zero_based_index
            : -1;
      orbitals.forEach(function (orb) {
        var opt = document.createElement("option");
        opt.value = orb.zero_based_index;
        opt.textContent =
          orb.label + " (" + Number(orb.energy_hartree).toFixed(3) + " Ha)";
        if (orb.zero_based_index === currentIdx) opt.selected = true;
        dom.$selectOrbital.appendChild(opt);
      });
      state.selectedOrbitalIndex = currentIdx;
      if (dom.$grpOrbitalSelect) dom.$grpOrbitalSelect.hidden = false;
      return;
    }

    if (!moE.length) {
      if (dom.$grpOrbitalSelect) dom.$grpOrbitalSelect.hidden = true;
      return;
    }

    var homoIdx = -1;
    for (var i = 0; i < moE.length; i++) {
      if (moO[i] > 0) homoIdx = i;
    }
    var lumoIdx = homoIdx >= 0 && homoIdx + 1 < moE.length ? homoIdx + 1 : -1;
    var info2 =
      (result.visualization && result.visualization.orbital_info) ||
      result.orbital_info ||
      {};
    var currentIdx2 =
      info2.orbital_index != null ? info2.orbital_index : homoIdx;
    var start = Math.max(0, homoIdx - 4);
    var end = Math.min(moE.length, (lumoIdx >= 0 ? lumoIdx : homoIdx) + 5);

    for (var j = start; j < end; j++) {
      var opt = document.createElement("option");
      opt.value = j;
      var label = "MO " + j;
      if (j === homoIdx) label = "HOMO";
      else if (j === lumoIdx) label = "LUMO";
      else if (j === homoIdx - 1) label = "HOMO-1";
      else if (j === homoIdx - 2) label = "HOMO-2";
      else if (lumoIdx >= 0 && j === lumoIdx + 1) label = "LUMO+1";
      else if (lumoIdx >= 0 && j === lumoIdx + 2) label = "LUMO+2";
      opt.textContent = label + " (" + Number(moE[j]).toFixed(3) + " Ha)";
      if (j === currentIdx2) opt.selected = true;
      dom.$selectOrbital.appendChild(opt);
    }
    state.selectedOrbitalIndex = currentIdx2;
    if (dom.$grpOrbitalSelect) dom.$grpOrbitalSelect.hidden = false;
  }

  function silentFetchOrbital(idx) {
    var result = state.result;
    if (!result) return;

    var molName = result.molecule_name || result.structure_name || "molecule";
    var method = result.method || "RHF";
    var basis = result.basis || result.basis_set || "sto-3g";
    var orbitals = result.orbitals || [];
    var selectedOrb = null;
    for (var i = 0; i < orbitals.length; i++) {
      if (orbitals[i].zero_based_index === idx) {
        selectedOrb = orbitals[i];
        break;
      }
    }
    var orbLabel = selectedOrb ? selectedOrb.label : "MO " + idx;

    showMiniSpinner(true);

    fetch("/api/compute/jobs", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        structure_query: molName,
        job_type: "orbital_preview",
        method: method,
        basis: basis,
        charge: result.charge || 0,
        multiplicity: result.multiplicity || 1,
        orbital: orbLabel,
      }),
    })
      .then(function (r) {
        if (!r.ok) throw new Error("Submit failed: " + r.status);
        return r.json();
      })
      .then(function (data) {
        var jobId = data.job_id || data.id;
        if (!jobId) throw new Error("No job_id");
        return pollJobResult(jobId);
      })
      .then(function (jobResult) {
        showMiniSpinner(false);
        var cubeB64 = findCubeB64(jobResult, "orbital");
        if (!cubeB64) return;

        // Cache
        var orbs = (state.result && state.result.orbitals) || [];
        for (var i = 0; i < orbs.length; i++) {
          if (orbs[i].zero_based_index === idx) {
            orbs[i].cube_data = cubeB64;
            break;
          }
        }

        var cubeStr = safeAtob(cubeB64);
        if (!cubeStr) return;

        clearViewer(state.viewer);
        addMoleculeModel(state.viewer, state.result);
        addOrbitalSurfaces(state.viewer, cubeStr);
        if (state.showLabels && state.model)
          addLabels(state.viewer, state.result);
        state.viewer.render();
        state.mode = "orbital";
        showControls("orbital");
        showOrbitalLegend();
      })
      .catch(function (err) {
        showMiniSpinner(false);
        console.error("[Viewer] Silent fetch failed:", err);
      });
  }

  function pollJobResult(jobId) {
    return new Promise(function (resolve, reject) {
      var count = 0;
      function tick() {
        fetch("/api/compute/jobs/" + jobId + "?include_result=true")
          .then(function (r) {
            return r.json();
          })
          .then(function (job) {
            var s = job.status || "";
            if (s === "completed" || s === "done")
              return resolve(job.result || job);
            if (s === "failed" || s === "error")
              return reject(new Error(job.error || "Job failed"));
            if (++count >= 60) return reject(new Error("Poll timeout"));
            setTimeout(tick, 500);
          })
          .catch(reject);
      }
      tick();
    });
  }

  /* ─────────────────────────────────────────
     MAIN RESULT HANDLER
     ───────────────────────────────────────── */
  function handleResult(detail) {
    var result = detail.result;
    var jobId = detail.jobId;
    var source = detail.source;

    if (!result) {
      if (state.viewer) {
        clearViewer(state.viewer);
        state.viewer.render();
      }
      state.result = null;
      state.jobId = null;
      state.mode = "none";
      if (dom.$empty) dom.$empty.hidden = false;
      if (dom.$controls) dom.$controls.hidden = true;
      hideLegend();
      return;
    }

    state.result = result;
    state.jobId = jobId;

    if (source === "history" && jobId) restoreViewerSnapshot(jobId);

    var promise;
    if (findCubeB64(result, "orbital")) {
      promise = renderOrbital(result);
    } else if (findCubeB64(result, "esp")) {
      promise = renderESP(result);
    } else if (getXyz(result)) {
      promise = renderMolecule(result);
    } else {
      state.mode = "none";
      if (dom.$empty) {
        dom.$empty.hidden = false;
        var t = dom.$empty.querySelector(".viewer-empty__text");
        if (t) t.textContent = "No visualization data for this result";
      }
      if (dom.$controls) dom.$controls.hidden = true;
      hideLegend();
      return;
    }

    if (promise)
      promise.then(saveViewerSnapshot).catch(function (e) {
        console.error("[Viewer] Render failed:", e);
      });
  }

  /* ─────────────────────────────────────────
     TRAJECTORY PLAYER
     ───────────────────────────────────────── */
  function initTrajectoryPlayer(result) {
    var $traj = document.getElementById("trajectoryPlayer");
    if (!$traj) return;
    var trajectory = result.trajectory;
    if (!trajectory || trajectory.length < 2) {
      $traj.hidden = true;
      state.trajectoryFrames = [];
      return;
    }
    state.trajectoryFrames = trajectory;
    state.trajectoryFrame = trajectory.length - 1;
    state.trajectoryPlaying = false;
    $traj.hidden = false;

    var $slider = document.getElementById("trajSlider");
    var $btnPlay = document.getElementById("trajPlay");
    var $btnPause = document.getElementById("trajPause");
    if ($slider) {
      $slider.min = 0;
      $slider.max = trajectory.length - 1;
      $slider.value = $slider.max;
      $slider.oninput = function () {
        goToTrajectoryFrame(parseInt($slider.value, 10));
      };
    }
    if ($btnPlay) $btnPlay.onclick = playTrajectory;
    if ($btnPause) $btnPause.onclick = pauseTrajectory;
    updateTrajectoryLabel();

    if (result.trajectory_xyz && state.viewer) {
      state.viewer.removeAllModels();
      state.viewer.addModelsAsFrames(result.trajectory_xyz, "xyz");
      state.viewer.setStyle(
        {},
        { stick: { radius: 0.15 }, sphere: { scale: 0.3 } },
      );
      state.viewer.zoomTo();
      state.viewer.render();
      if (trajectory.length > 0) {
        state.viewer.setFrame(trajectory.length - 1);
        state.viewer.render();
      }
    }

        pauseTrajectory();
        return;
      }

function fetchHistory() {
  return fetch(PREFIX + "/compute/jobs?include_result=true")
    .then(function (res) {
      if (!res.ok) return;
      return res.json();
    })
    .then(function (data) {
      if (!data) return;
      var jobs = Array.isArray(data) ? data : data.items || data.jobs || [];
      jobs.sort(function (a, b) {
        return (a.created_at || 0) - (b.created_at || 0);
      });
      jobs.forEach(function (j) {
        App.upsertJob(j);
      });
      renderHistory();

      // FIX #17: 활성 job이 없으면 마지막 완료 job을 자동 선택
      if (!App.store.activeJobId && App.store.jobOrder.length > 0) {
        // 최신 완료 job 찾기
        for (var i = App.store.jobOrder.length - 1; i >= 0; i--) {
          var jid = App.store.jobOrder[i];
          var j = App.store.jobsById[jid];
          if (
            j &&
            (j.status === "completed" || j.status === "done") &&
            j.result
          ) {
            App.setActiveJob(jid);
            break;
          }
        }
      }
    })
    .catch(function (e) {
      console.error("fetchHistory error:", e);
    });
}

## 체크리스트 검증 결과

### ✅ 1. `renderOrbital`에서 `scheme` 변수 선언 및 `orbNegative` 참조

수정 코드에서는 `renderOrbital`이 직접 isosurface를 추가하지 않고, `addOrbitalSurfaces(viewer, cubeStr)`를 호출합니다. `addOrbitalSurfaces` 내부에서 `var scheme = getScheme();`로 선언하고 `scheme.orbPositive`, `scheme.orbNegative`를 사용합니다. 원래 결함이었던 미선언 `scheme` 참조는 완전히 해소되었습니다.

---

### ✅ 2. `_vizSwitchLock` 이중 선언 제거

수정 코드에서는 `_vizSwitchLock` 변수가 아예 존재하지 않습니다. `switchVizMode`가 `renderOrbital`/`renderESP`의 Promise를 직접 호출하는 구조로 바뀌었기 때문에 lock 메커니즘 자체가 불필요해졌습니다. `state.mode === newMode` 가드가 재진입을 방지합니다. 통과합니다.

---

### ✅ 3. `xyz_block` 부재 시 폴백 (atoms→XYZ, cube 추출)

`getXyz(result)` 함수가 `viz.xyz`, `viz.molecule_xyz`, `viz.xyz_block`, `result.xyz_block`, `result.xyz`를 순서대로 탐색하고, 모두 없으면 `result.atoms`가 있을 때 `buildXyzFromAtoms(atoms)`를 호출합니다. `renderOrbital`에서는 `addMoleculeModel`이 실패한 경우(`!state.model`) cube 데이터에서 모델을 추출하는 `viewer.addModel(cubeStr, "cube")` 폴백이 존재합니다. 통과합니다.

---

### ✅ 4. Orbital Selector가 `mo_energies`/`mo_occupations` 존재 시 렌더링

`populateOrbitalSelector`에서 `result.orbitals`가 없을 경우 `result.mo_energies`를 확인하고, `moE.length > 0`이면 HOMO 인근 MO 목록을 생성합니다. `dom.$grpOrbitalSelect.hidden = false`로 UI를 표시합니다. 통과합니다.

---

### ✅ 5. History 항목 이름 폴백 순서 및 UUID 미노출

`app.js`의 `getJobDisplayName`에서 순서대로 확인합니다: `user_query` → `molecule_name`/`molecule`/`structure_name`/`structure_query` (payload 포함) → `method`/`basis` 조합. 마지막 폴백은 `job_type`을 사람이 읽을 수 있는 형태로 변환하는 것이지, `job_id` UUID를 그대로 노출하지 않습니다. 통과합니다.

### ❌ 6. History 클릭 → `setActiveJob()` → viewer + results 동기화 흐름

**문제**: `app.js`의 히스토리 클릭 핸들러는 `App.setActiveJob(jobId)`를 호출하고 `renderHistory()`를 다시 부릅니다. 그런데 `setActiveJob`의 구현이 이 코드 덤프에 없습니다. `QCVizApp` 코어(아마 별도 파일)에 정의되어 있다고 가정하면, 해당 함수가 `result:changed` 이벤트를 emit해야 viewer가 반응합니다.

하지만 수정된 `viewer.js`의 `handleResult`는 `App.on("result:changed", handleResult)`로 바인딩되어 있고, `handleResult`는 `detail.result`, `detail.jobId`, `detail.source`를 기대합니다. `setActiveJob`이 이 형태로 이벤트를 발행하는지 확인할 수 없습니다.

또한 `viewer.js`에는 `App.on("result:switched", handleResultSwitched)`도 바인딩되어 있습니다. `setActiveJob`이 `result:changed`를 쏘는지 `result:switched`를 쏘는지에 따라 동작이 달라집니다.

**수정 제안**: `app.js`의 히스토리 클릭 핸들러를 보강하여, `setActiveJob` 호출 후 명시적으로 `result:changed` 이벤트를 발행합니다.

```javascript
// History click — 수정됨
if ($historyList) {
  $historyList.addEventListener("click", function (e) {
    var item = e.target.closest(".history-item");
    if (!item) return;
    var jobId = item.dataset.jobId;
    if (!jobId) return;

    App.setActiveJob(jobId);

    // setActiveJob이 result:changed를 emit하지 않을 경우를 대비하여
    // 명시적으로 viewer에 결과를 전달
    var job = App.store.jobsById[jobId];
    var result =
      (job && job.result) ||
      (App.store.resultsByJobId && App.store.resultsByJobId[jobId]) ||
      null;
    if (result) {
      App.emit("result:changed", {
        result: result,
        jobId: jobId,
        source: "history",
      });
    }

    renderHistory();
  });
}
```

### ❌ 9. `chat.js` — `[object Object]` 방지 재귀 파싱

**문제**: 역시 `chat.js` 미제공으로 검증 불가. 에러 메시지 표시 함수에 적용해야 할 유틸리티 함수를 제공합니다.

```javascript
// chat.js에 추가할 유틸리티
function stringifyError(val) {
  if (val == null) return "";
  if (typeof val === "string") return val;
  if (val instanceof Error) return val.message || String(val);
  if (typeof val === "object") {
    // 중첩 error/message/detail 필드 우선 탐색
    if (val.message && typeof val.message === "string") return val.message;
    if (val.error && typeof val.error === "string") return val.error;
    if (val.detail && typeof val.detail === "string") return val.detail;
    // 재귀 시도
    if (val.error && typeof val.error === "object")
      return stringifyError(val.error);
    if (val.detail && typeof val.detail === "object")
      return stringifyError(val.detail);
    // 최종 fallback: JSON
    try {
      return JSON.stringify(val, null, 2);
    } catch (_) {
      return String(val);
    }
  }
  return String(val);
}

// 사용 예: 에러 메시지 표시 시
// 기존: appendMessage("error", data.error);  ← [object Object] 가능
// 수정: appendMessage("error", stringifyError(data.error));
```

### 신규 결함 A: `viewer.js` — `getXyz` 함수의 논리 오류

```javascript
return (
  viz.xyz ||
  viz.molecule_xyz ||
  viz.xyz_block ||
  result.xyz_block ||
  result.xyz ||
  null ||
  (result.atoms && result.atoms.length ? buildXyzFromAtoms(result.atoms) : null)
);
```

`|| null ||` 이후의 `buildXyzFromAtoms` 호출은 **절대 도달하지 않습니다**. `null`은 falsy이므로 다음 피연산자로 넘어가긴 하지만, 문제는 `result.xyz`가 빈 문자열 `""`일 때입니다. 빈 문자열은 falsy이므로 통과하지만, 실제 논리적으로 `result.xyz`가 truthy가 아닌 경우에만 atoms 폴백에 도달해야 합니다. 현재 코드는 **동작은 합니다** (null은 falsy니까 다음으로 넘어감), 하지만 가독성이 매우 나쁘고 의도가 불명확합니다.

```javascript
// 수정된 getXyz
function getXyz(result) {
  if (!result) return null;
  var viz = result.visualization || {};
  var xyz =
    viz.xyz ||
    viz.molecule_xyz ||
    viz.xyz_block ||
    result.xyz_block ||
    result.xyz ||
    null;
  if (!xyz && result.atoms && result.atoms.length) {
    xyz = buildXyzFromAtoms(result.atoms);
  }
  return xyz;
}
```

이 수정은 원래 **구 코드**에는 올바르게 되어 있었는데 (분리된 `if (!xyz && ...)` 구문), 재작성 과정에서 한 줄에 합치면서 오류가 발생했습니다.

### 신규 결함 B: `viewer.js` — `switchVizMode`의 `state.mode = "switching"` 가드 불완전

```javascript
function switchVizMode(newMode) {
  if (!state.result) return;
  if (state.mode === newMode) return;
  state.result && (state.mode = "switching");
  // ...
}
```

`state.result && (state.mode = "switching")`은 `state.result`가 이미 첫 줄에서 체크되었으므로 조건이 항상 true입니다. 그러나 더 큰 문제는, `renderOrbital`/`renderESP`가 비동기(Promise)인데, 그 사이에 다른 `switchVizMode` 호출이 들어오면 `state.mode`가 `"switching"`이라서 `=== newMode` 가드에 걸리지 않고 통과해버립니다. 즉, 빠르게 ESP→Orbital→ESP를 누르면 중첩 렌더가 발생할 수 있습니다.

```javascript
// 수정된 switchVizMode
var _switching = false;

function switchVizMode(newMode) {
  if (!state.result) return;
  if (state.mode === newMode) return;
  if (_switching) return; // 전환 중 재진입 방지

  _switching = true;
  var renderFn =
    newMode === "orbital"
      ? renderOrbital
      : newMode === "esp"
        ? renderESP
        : null;

  if (!renderFn) {
    _switching = false;
    return;
  }

  renderFn(state.result)
    .then(function () {
      saveViewerSnapshot();
    })
    .catch(function (e) {
      console.error("[Viewer] switchVizMode error:", e);
    })
    .then(function () {
      _switching = false; // finally 역할
    });
}
```

### 신규 결함 C: `app.js` — `html`이 `$historyEmpty.insertAdjacentHTML` 전에 비어있을 수 있음

`renderHistory()`에서 `filtered.length === 0`이면 early return하지만, `filtered.length > 0`일 때 `html` 변수에 모든 아이템을 concat하고 `$historyEmpty` 존재 여부에 따라 `insertAdjacentHTML` 또는 `innerHTML`로 삽입합니다. 문제는 `$historyEmpty`가 존재하면 `$historyEmpty.insertAdjacentHTML("beforebegin", html)`을 사용하는데, 이 방식은 `$historyEmpty` 앞에 **누적**으로 삽입합니다. `renderHistory` 시작부에서 기존 `.history-item`을 제거하므로 실질적으로 문제없지만, DOM 조작 순서가 fragile합니다.

이 부분은 현재 동작에 문제가 없으므로 **경고 수준**입니다. 더 안전한 패턴은 `DocumentFragment`를 사용하는 것입니다.

## 6. History 클릭 → viewer 동기화 — ❌ 부분 미반영

**문제**: 1차 검증에서 이 항목에 대해 수정 코드를 제공했지만, 해당 수정은 `app.js`의 **히스토리 클릭 핸들러만** 보강한 것입니다. 실제로 완전한 동기화 흐름을 보장하려면, `App.setActiveJob` 자체의 구현이 `result:changed`를 emit해야 하거나, 아니면 모든 호출부(히스토리 클릭, 세션 탭 클릭, fetchHistory 자동 복원)에서 일관되게 emit해야 합니다. 세션 탭 클릭 핸들러에도 동일한 보강이 누락되어 있습니다.

```javascript
// app.js — 세션 탭 클릭 핸들러 (수정)
if ($sessionTabs) {
  $sessionTabs.addEventListener("click", function (e) {
    var tab = e.target.closest(".session-tab");
    if (!tab) return;
    var jid = tab.getAttribute("data-job-id");
    if (!jid || jid === App.store.activeJobId) return;

    App.setActiveJob(jid);

    var job = App.store.jobsById[jid];
    var result =
      (job && job.result) ||
      (App.store.resultsByJobId && App.store.resultsByJobId[jid]) ||
      null;
    if (result) {
      App.emit("result:changed", {
        result: result,
        jobId: jid,
        source: "session-tab",
      });
    }
  });
}
```

또한 `fetchHistory`의 자동 복원 부분(`App.setActiveJob(jid)` 호출)에도 명시적 emit이 필요합니다:

```javascript
// app.js — fetchHistory 내 자동 복원 부분 (수정)
if (!App.store.activeJobId && App.store.jobOrder.length > 0) {
  for (var i = App.store.jobOrder.length - 1; i >= 0; i--) {
    var jid = App.store.jobOrder[i];
    var j = App.store.jobsById[jid];
    if (j && (j.status === "completed" || j.status === "done") && j.result) {
      App.setActiveJob(jid);
      App.emit("result:changed", {
        result: j.result,
        jobId: jid,
        source: "history",
      });
      break;
    }
  }
}
```

위 3곳(히스토리 클릭, 세션 탭 클릭, fetchHistory 자동 복원) 모두에서 `App.emit("result:changed", ...)`을 호출해야 viewer.js의 `handleResult`가 동작하여 뷰어·결과 탭·레이블이 동기화됩니다. 이상적으로는 `App.setActiveJob` 내부에 이 emit을 넣는 것이 가장 깔끔하지만, 해당 함수의 소스가 제공되지 않았으므로 호출부에서 보장합니다.

function stringifyError(val) {
  if (val == null) return "";
  if (typeof val === "string") return val;
  if (val instanceof Error) return val.message || String(val);
  if (typeof val === "object") {
    if (typeof val.message === "string") return val.message;
    if (typeof val.error === "string") return val.error;
    if (typeof val.detail === "string") return val.detail;
    if (typeof val.error === "object") return stringifyError(val.error);
    if (typeof val.detail === "object") return stringifyError(val.detail);
    try {
      return JSON.stringify(val, null, 2);
    } catch (_) {
      return String(val);
    }
  }
  return String(val);
}

### 신규 결함 E: `viewer.js` — `init()` 내 `ensureViewer` 성공 경로에서 `result:changed` 이벤트와의 경쟁

`init()`에서 `ensureViewer().then(...)` 내부에서 `App.store.resultsByJobId[activeJobId]`를 찾아 직접 `renderOrbital`/`renderESP`/`renderMolecule`을 호출합니다. 동시에 `app.js`의 `fetchHistory`가 `App.setActiveJob`을 호출하여 `result:changed`를 emit할 수 있습니다. 두 경로가 거의 동시에 실행되면 동일한 결과에 대해 렌더링이 2번 발생합니다.
