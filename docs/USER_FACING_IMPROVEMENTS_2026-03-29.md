# User-Facing Improvements Summary

## What Users Will Notice

- `베 ㄴ젠`, `베ㄴ젠`처럼 띄어쓰기나 한글 자모가 깨진 입력도 예전보다 훨씬 잘 `benzene`으로 복원됩니다.
- `니트로 벤젠`, `니트로벤젠` 같은 한국어 치환기+모체 표현도 이제 하나의 분자로 이해하고 바로 계산으로 이어집니다.
- 분자명을 잘못 이해했을 때 뜬금없이 `water` 같은 기본 추천으로 튀는 일이 크게 줄었습니다.
- `Biphenyl`, `benzoic acid`, `아세트산`, `메틸아민` 같은 단일 분자 요청은 불필요한 후보 선택창 없이 바로 계산으로 들어가는 경우가 많아졌습니다.
- 같은 세션에서 이미 분자를 한 번 계산했다면, 이후 `ESP도 그려줘`, `HOMO LUMO ESP가 궁금`, `ESP ㄱㄱ` 같은 후속 질문은 그 분자를 이어받아 처리합니다.
- 반대로 이전 문맥이 없는데 `ESP도`, `HOMO 보여줘`처럼 분석만 던지면, 엉뚱한 분자를 추측하지 않고 어떤 분자인지 더 정확하게 다시 묻습니다.
- clarification이 필요한 경우에도 raw 문장 전체를 이상한 분자 후보로 올리지 않고, 더 그럴듯한 후보만 보여주도록 정리됐습니다.
- `EMIM+ TFSI-` 같은 이온쌍 입력은 단일 분자처럼 오해하지 않고, 양이온/음이온 조합으로 인식하는 방향으로 안정화됐습니다.
- 웹소켓/실시간 채팅 경로 테스트를 더 열어놔서, REST와 실시간 채팅에서 동작 차이가 나는 위험도 줄었습니다.
- 전체적으로 사용자가 느끼는 변화는 한 줄로 정리하면, `덜 엉뚱하게 알아듣고, 덜 많이 되묻고, 이전 문맥을 더 잘 이어받는다`입니다.
